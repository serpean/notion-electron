# Notion on Linux
> Notion embedded on a Electron application.

## Installation

Linux:

Change `x` by your version.

```sh
sudo dpkg -i ~/Downloads/notion-electron_0.0.x_amd64.deb
```

## Development setup


```sh
npm/yarn install
npm/yarn start
```
